# ✅ Testing y Validación

## Validación de Datos

- **Cliente**: Validación de formularios con Ant Design
- **Servidor**: Validación en repositorios y servicios
- **Tipos**: TypeScript para type safety

## Manejo de Errores

- Try-catch en operaciones asíncronas
- Mensajes de error user-friendly
- Logging de errores en consola (desarrollo)

## Validación de Roles y Permisos

- Utilidades en `/utils/auth/permissions.utils.ts`
- `hasPermission(role, permission)` para verificar permisos
- Componente `RequireRole` para protección de UI
